================================================================================
COMMON ISSUES & HOW TO VERIFY THEY'RE FIXED
================================================================================

ISSUE: Horizontal scroll on mobile (should NOT happen)
  FIX: Check grid and flex layouts are single-column, no min-width on cards
  TEST: Resize to 375px, drag horizontally → should only scroll intentionally
        (e.g., pipeline, tables with overflow-x: auto)

ISSUE: Text too small to read (should be 12px+)
  FIX: Check all p, span, labels are 12px minimum
  TEST: Hold phone at arm's length, text should be readable without zoom

ISSUE: Buttons too small to tap (should be 44px+)
  FIX: Check all buttons have min-height: 44px, min-width: 44px
  TEST: Use Chrome DevTools → triple-click button, measure with inspect tool

ISSUE: Form inputs squished or overlapping
  FIX: Check inputs are 100% width, padding 10px+, height 44px+
  TEST: Try typing in mobile view, input should not be cramped

ISSUE: Modal goes off-screen
  FIX: Check modal width: 90%, max-width: 90%, max-height: 90vh
  TEST: Open any modal on 320px screen, should be fully visible

ISSUE: Sidebar/navigation takes up too much space
  FIX: Check sidebar hidden by default, hamburger menu visible
  TEST: Tap hamburger icon, drawer should slide from left (or nav bottom)

ISSUE: Pipeline columns squish together on mobile
  FIX: Check pipeline columns min-width: 280px, flex-shrink: 0
  TEST: Open CompanyATS on mobile, swipe horizontal, columns stay 280px
